# Smart City Information - Darbhanga

This project is a web/mobile application designed to provide comprehensive, real-time information about various urban services and facilities in Darbhanga city. It aims to improve citizen engagement, streamline access to public resources, and promote smart city initiatives.

## Features
- Information on essential services like hospitals, police stations, and government offices
- Maps and navigation support within Darbhanga city
- User feedback and reporting system for city maintenance issues

## Tech Stack
- Frontend: HTML, CSS
